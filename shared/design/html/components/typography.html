Typography

Bitmap font banks 132–136. Fonts are rendered with a fixed character advance (the width argument to DrawText()); glyph height is the hit-test value used by Overlay::MouseInside(). Sample text is drawn here in the VT323 web font as a near-pixel monospace substitute — the actual bitmap glyphs live in shared/assets/bin_spr/SPR_132.BIN–SPR_136.BIN.

Notes

  • Glyph N in bank B = sprite index (asciiCode - asciiOffset) where asciiOffset = 34 for bank 132 and 33 for the rest.
  • Space (0x20) is never indexed — the renderer just advances the cursor.
  • No extended-ASCII / Unicode coverage; ASCII 33–127 only.
  • Per-glyph offsetY handles descenders (g, p, y).
