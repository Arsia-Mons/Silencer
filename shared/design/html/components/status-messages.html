Status Messages (DrawStatus)

Stack of short event messages near bottom-center (kills, pickups). Each message centered at x = (640 - strlen*7)/2, base Y = 370. Messages stack upward (yoffset −= 10 per line). Newest at the bottom. Last 16 ticks fade brightness 128 → 0.

silencer fragged Bob
Alice picked up the secret
Carol joined Team Lazarus
Eve dropped the secret

Pseudocode

baseY = 370
font: bank 133, width 7
yoffset = 0
for each message (newest first):
    color = stored per-message
    time  = stored per-message
    brightness = 128
    if time <= 16:
        brightness -= (16 - time) * 8
    brightness2 = max(brightness - 64, 8)
    drawText(s, x+1, baseY+yoffset+1, text, 133, 7, color, brightness2)
    drawText(s, x,   baseY+yoffset,   text, 133, 7, color, brightness)
    yoffset -= 10
