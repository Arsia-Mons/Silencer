Per-character brightness animation pseudocode

for each character i in message:
    brightness = 128
    if message_i - messagetime + 8 >= 0:
        brightness -= (message_i - messagetime + 8) * 8     // fade out
    if message_i % 32 >= 16:
        brightness += (16 - (message_i % 16)) * 2           // triangular wave
    else:
        brightness += (message_i % 16) * 2
    if message_i - i <= 5:
        brightness += 40 - ((message_i - i) * 8)            // typewriter boost
