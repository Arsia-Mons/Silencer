Toggle

Two visual modes — agency icon (sprite bank 181) and checkbox (sprite bank 7 indices 18/19). Toggles with the same non-zero set form a radio group: selecting one deselects the others.

Agency mode (radio group, set = 1)

5 agency toggles, 42 px apart (lobby Character panel)
Selected: brightness 128 (neutral) + tint 112 (#549C68).
Deselected: brightness 32 (ch * 0.25) — very dark.
