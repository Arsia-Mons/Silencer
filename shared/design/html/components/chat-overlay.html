In-Game Chat Overlay

Background rect: (400, 280) 231 × 30. Built on the horizontal-stretch panel (bank 188). Up to 5 visible lines, hard-truncated to 36 chars each, line spacing 10 px. Font: bank 133, advance 6, brightness 136 (slightly bright).

[in-game viewport — faded]
[All] hv: nice push
[Team] alice: regrouping at A
[Sys] enemy secret dropped
(ALL):

Pseudocode

rect = {x:400, y:280, w:231, h:30}
drawMessageBackground(surface, rect)

yoffset = 10
for each chatline (max 5):
    text = truncate(line, 36)
    drawText(surface, rect.x+10, rect.y+yoffset, text,
             bank=133, width=6, color=0, brightness=136)
    yoffset += 10

if chat input is active:
    prepend = "(ALL):" or "(TEAM):"
    drawText(surface, rect.x+10, rect.y+yoffset, prepend, 133, 6, 0, 136)
    textInput.x = rect.x + strlen(prepend)*6 + 10
    textInput.y = rect.y + yoffset
    drawTextInput(surface, textInput)
