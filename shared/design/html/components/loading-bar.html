Loading Bar

The simplest possible progress indicator. No border, no text, no background — just a 500 × 20 px filled rectangle in palette index 123 (#949494, mid-gray) that grows as progress / totalItems increases. Centered on (320, 240) of the 640 × 480 buffer. Update rate throttled to every 100 ms.

Pseudocode

totalWidth   = 500
filledWidth  = (progress / totalItems) * totalWidth
color        = 123    // #949494
x1 = (640 - totalWidth) / 2     // 70
y1 = (480 - 20) / 2             // 230
x2 = (640 + filledWidth) / 2
y2 = (480 + 20) / 2             // 250
drawFilledRectangle(surface, x1, y1, x2, y2, color)
