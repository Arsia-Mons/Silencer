Horizontal-Stretch Panel (sprite bank 188)

A two-row panel — only the top row (sprites 0/1/2) and bottom row (sprites 6/7/8) are ever drawn. Indices 3, 4, 5 (left edge / center / right edge) are reserved but never blitted, so the panel has no vertical fill. Right-side corners are placed at x + width − 36.

Pseudocode (from §Horizontal-Stretch Panel)

// Top row
blit(bank188[0], x=rect.x, y=rect.y)
tileX = spriteWidth[188][0]
while tileX < rect.w - spriteWidth[188][2]:
    clipW = min(rect.w - tileX - 36, spriteWidth[188][1])
    blit(bank188[1], x=rect.x + tileX, y=rect.y, srcW=clipW)
    tileX += clipW
blit(bank188[2], x=rect.x + rect.w - 36, y=rect.y)
// Bottom row identical, using indices 6/7/8.
